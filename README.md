# InventoryApp

Technologies Used

Frontend:

Vue 3 – enables a reactive interface and efficient state management.
Tailwind CSS – facilitates rapid development of a responsive and modern user interface.
Google reCAPTCHA v2 – protects the contact form from spam and automated submissions.
HTML5 & CSS3 – provide the foundational structure and styling for the application.

Backend:

Node.js & Express – handle server-side logic and RESTful API endpoints.
PostgreSQL (via pg-promise) – stores product data and generates inventory statistics.
Axios – manages external requests, including CAPTCHA verification.
CORS – ensures secure communication between frontend and backend.
File system (fs) – stores contact messages in a persistent text file.

Plugins & Libraries:

Tailwind CSS CDN – allows rapid styling without build tools.
Vue 3 CDN – integrates the reactive frontend framework efficiently.
Google reCAPTCHA – safeguards the contact form against bots.

Використані технології: 
Фронтенд:

Vue 3 
Tailwind CSS 
Google reCAPTCHA v2 - захист контактної форми від спаму. 
HTML5 & CSS3 - базова структура та стилізація. 

Бекенд: 

Node.js + Express - серверна логіка та REST API. 
PostgreSQL (через pg-promise) - зберігання даних про товари та статистику. 
Axios - робота з зовнішніми запитами (CAPTCHA перевірка). 
CORS - забезпечення безпечного обміну даними між фронтендом і бекендом. 
Файлова система (fs) - збереження контактних повідомлень у текстовий файл. 

Плагіни та бібліотеки: 

Tailwind CSS CDN - для швидкого застосування стилів. 
Vue 3 CDN - підключення реактивного фреймворку. 
Google reCAPTCHA - захист від ботів у формі контактів.
